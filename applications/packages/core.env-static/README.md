## core.env-static
Library to handle environment based constants.

Exports:
* `CLIENT_STATIC`
* `SERVER_STATIC`
* `COMMON_STATIC`

### Usage
```ts

import {
    CLIENT_STATIC,
    SERVER_STATIC,
    COMMON_STATIC
} from "@wraithlight/core.env-static";

// Read any properties here from the imported objects.

```
